# Project Manager - Landing Page

A professional landing page showcasing the Construction Project Manager application, designed for recruiters and hiring managers.

## Features

- **Modern Design**: Clean, professional UI with smooth animations
- **Responsive**: Works perfectly on all devices
- **Feature Showcase**: Highlights key application capabilities
- **Tech Stack Display**: Shows modern technologies used
- **No Dependencies**: Pure HTML, CSS, and JavaScript

## Usage

Simply open `index.html` in a web browser or deploy to any static hosting service:

- GitHub Pages
- Netlify
- Vercel
- Any web server

## Customization

1. **Update GitHub Links**: Replace `yourusername/project_manager` with your actual GitHub username/repo
2. **Add Screenshots**: Consider adding screenshots in a `images/` folder
3. **Customize Colors**: Modify CSS variables in `styles.css`
4. **Add Demo Video**: Embed a demo video in the demo section

## Files

- `index.html` - Main landing page
- `styles.css` - All styling
- `script.js` - Interactive features and animations
- `README.md` - This file
