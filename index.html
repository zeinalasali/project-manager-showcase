Key Features

AI Copilot with RAG

Intelligent assistant powered by GPT-5 Nano with Retrieval Augmented Generation (RAG). Uses LangChain and pgvector for semantic search across your project data, providing context-aware insights and answers.

Project & Item Management

Organize construction projects with detailed items, track status, budget vs actual revenue, and monitor progress from start to completion.

Cost Tracking

Track all project costs including materials, labor, equipment, and subcontractors. Automatic labor cost calculation based on hourly rates.

Profitability Analysis

Real-time profitability tracking comparing actual revenue to total costs. Get insights on project and item-level profitability with visual indicators.

Role-Based Access

Secure multi-tenant system with admin and user roles. Invitation key system for controlled user onboarding with predefined roles and hourly rates.

Row-Level Security

Enterprise-grade security with PostgreSQL RLS policies ensuring users only access data from their organization. Secure API with service role authentication.

Retrieval Augmented Generation (RAG)

Combines vector similarity search with LLM reasoning. Uses LangChain to retrieve relevant project context and generate intelligent responses based on your actual data. LangChain orchestrates the entire RAG pipeline from retrieval to generation.

pgvector & Vector Similarity

pgvector PostgreSQL extension for storing 1536-dimensional embeddings directly in your database. HNSW indexing enables fast semantic search to find similar project items based on meaning, not just keywords. All vector operations happen natively in PostgreSQL.

pgvector HNSW Index 1536-dim Vectors PostgreSQL Native

Automatic Embedding Generation

Every project item automatically generates embeddings using OpenAI API (text-embedding-3-small model). Embeddings stored in pgvector for instant semantic search. No manual backfill needed - your data is instantly searchable.
